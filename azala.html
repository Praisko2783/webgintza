Intentzio onenarekin

Ikastaro honetan zerotik abiatuko gara, ez da aldez aurretik programazioaz ezer jakin behar, ikasteko gogoa besterik ez da behar. Hauexek dira ikastaroaren lerro nagusiak:

  • HTML5, CSS3 eta JavaScript irakastea, haien xehetasunak aztertuz, ikasiz eta erabiliz.
  • Zalantzak argitzeko tokiak eta helbideak non diren erakustea.
  • Mundu errealeko profesionalek nola lan egiten duten erakustea, haien tresnak aurkeztuz, hala nola Github, Kode editoreak, nabigatzaileen xehetasunak edota zerbitzari lokalak.

Programazioaren munduan arazo bera konpontzeko modu asko daude, guztiak balekoak. Halere, ikastaroaren helburua ez da problema bat ebazteko modu guzti-guztiak ikastea (hori urteetako praktikarekin ikasten da)..

1. Atala

Internet

Internet gure bizitzan funtsezko bihurtu da, eta webgunea da interneteko teknologiak konektatzen dituena.

2. Atala

HTML

HTMLk sortutako egitura dokumentuaren funtsezko zatia izaten jarraitzen du dokumentuaren barruko espazioa definitzeko.

3. Atala

CSS

CSSk HTML elementuak jantzi egiten ditu, koloreak, letra motak, tamainak, animazioak... bezalako estiloak esleituz.

4. Atala

Diseinua

Nabigatzaileak kutxa batean kokatzen du HTML elementua. Elementuok 'Block' edo 'Inline' motakoak izan daitezke.

5. Atala

'Responsive'

Web-orria pantailari egokitzeko 'Media Queries' izeneko tresnaz baliatuko gara. Tresna hori CSSk hornitzen du.

6. Atala

Bideoak

Ikus-entzunezkoak komunikazio bide eraginkorra direnez, HTML5ek bideoak kudeatzeko elementu bat eskaintzen du.

7. Atala

JS

JavaScriptek jarraibideak modu sekuentzialean exekutatzen ditu, sistemari adierazteko zer nahi dugun egin.

8. Atala

AJAX

'XMLHttpRequest' objektua da JavaScript-etik informazioa lortzeko, orri osoa berriro kargatu beharrik gabe.

9. Atala

API Javascript (1)

Form (formularioak kudeatzeko eta balioesteko), Stream(Aparatuaren baliabideekin konektatzeko), Fullscream (pantaila osorik erabiltzeko), Canvas (marrazkiak, grafikoak eta animazioak egiteko).

10. Atala

API Javascript (2)

Pointer Lock (xaguaren funtzioak kontrolatzeko), Web Storage (datu gutxi batzuk gordetzeko), IndesedDB (datu ugari gorde beharra daukagunerako),File (erabiltzailearen fitxategiak erabili ahal izateko).

11. Atala

API Javascript (3)

Drag and Drop (pantailako elementuak arrastaka mugitzeko), Geolocation(erabiltzailearen kokapena erabiltzeko), History (erabiltzaileak zer bisitatu duen), Page Visibility (baliabide gastua kudeatzeko).

12. Atala

API Javascript (4)

Web Messaging messaging (aplikazioak bata bestearekin batzeko), Web Socket(joan-etorriko komunikazioak), Web RT (bapaeko komunikazioak), Web Audio (audioa kudeatzeko), Web Workers (albo prozesuak garatzeko).
