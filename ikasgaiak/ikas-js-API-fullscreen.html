Webgunea multimedia eta asmo anitzeko plataforma bihurtu da, eta posiblea. Hainbeste aplikazio berrirekin, nabigazio hitzak esanahia galdu du. Las nabigatzailearen leihoko tresnak jada ez dira beharrezkoak batzuetan erabiltzailearen eta aplikazioaren artean jartzen dira. Horregatik, nabigatzaileek API Fullscreen dute

Pantaila osoa

Fullscreen APIak aukera ematen digu dokumentuan edozein elementu hedatzeko, harik eta pantaila osoa. Ondorioz, nabigatzailearen interfazea ezkutatu egiten da eta erabiltzaileari aukera ematen dio gure bideo, irudi, aplikazio edo bideojokoetan arreta jartzea.

APIak propietateak, metodoak eta gertaerak ematen ditu elementu bat pantaila osoa, modu horretan irtetea, eta elementuari eta dokumentuari buruzko informazioa prozesuan parte hartzen ari dira.

  • fullscreenElement — Ezaugarri honek erreferentzia bat itzultzen dio pantaila osoan erakusten. Elementurik ez badago pantaila osoan, null balioa itzultzen du.
  • fullscreenEnabled — Hau boolear propietate bat da, itzultzean dokumentua pantaila osoa aktiba dezakezu edo, bestela, huts egin.
  • requestFullscreen () — Metodo hau dokumentuaren elementu guztiei aplikatzen zaie. Elementurako pantaila osoaren modua aktibatzen du metodoak.
  • exitFullscreen () — Metodo hau dokumentuari aplikatzen zaio. Elementu bat pantaila osoa, metodo honek bertan behera uzten du modua eta fokua berriro itzultzen du nabigatzailearen leihoa.

    • APIak honako ekitaldi hauek ere eskaintzen ditu.

  • fullscreenchange — Dokumentuak gertaera hau abiarazten du elementu bat sartzen denean edo pantaila osoaren modutik ateratzen da.
  • fullscreenerror — Gertaera hau elementuaren bidez abiarazten da, errorerik izanez gero (pantaila-modu osoa ez dago erabilgarri elementu horretarako edo dokumenturako).
