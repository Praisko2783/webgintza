Gida didaktiko orokorra

Gaitasun profesionalak

Titulazio honen jabea gai izango da:
  1. Marka eta web-estandarren lengoaiak erabiltzea, web-aplikazioen interfazeak garatzeko.
  2. Berariazko hizkuntza-tresnak eta -lengoaiak erabiltzea, espezifikazioei jarraituz, multimedia-osagaiak garatzeko.
  3. Interfaze baten elkarreragina, irisgarritasuna eta erabilgarritasuna ebaluatzea, aurrez ezarritako irizpideak egiaztatuz, aplikazio baten interfazean multimedia-osagaiak integratzeko.
  4. Baliabide informatikoen mekanizazioa kudeatzeko jarduerak programatzea eta abiaraztea.
  5. Irisgarritasun unibertsalari eta guztiontzako diseinuari erantzun egokia emateko beharrezkoak diren lanbide-ekintzak identifikatu eta proposatzea.
  6. Ikaskuntza-prozesuan egindako lanetan eta jardueretan kalitate-parametroak identifikatzea eta aplikatzea, ebaluazioaren eta kalitatearen kultura baloratzeko eta kalitatea kudeatzeko prozedurak gainbegiratzeko eta hobetzeko gai izateko.

Helburu orokorrak

  1. Web aplikazioetan interfazeak garatzeko estilo-eskuliburuaren arabera, markatzeko lengoaiak eta web-estandarrak erabilita.
  2. Web aplikazioetan integratuko diren multimedia-osagaiak garatzeko, eta, horretarako, tresna espezifikoak erabiltzeko eta ezarritako zehaztapenak betetzeko.
  3. Web aplikazio baten interfazean multimedia-osagaiak integratzeko, eta aplikazioaren interaktibitatea, erabilerraztasuna eta usagarritasuna analizatzeko.
  4. Web zerbitzariaren ingurunean software osagaiak garatzeko eta integratzeko, tresna eta lengoaia espezifikoak erabilita, eta, hala, aplikazioaren zehaztapenak betetzeko.
  5. Zerbitzuak garatzeko eta haien funtzioak beste web aplikazio batzuetan integratzeko, funtzionaltasuna ziurtatuta.
  6. Web aplikazioetan zerbitzu eta eduki banatuak integratzeko, eta haien funtzionaltasuna ziurtatzeko.
    7. --------------
  7. Markatzeko lengoaiak eta web-estandarrak erabiltzea, web-aplikazioetan interfazeak garatzeko.
  8. Lengoaiak, objektuak eta tresnak erabiltzea, eta garatzeko zehaztapenak interpretatzea.
  9. Lengoaiak, objektuak eta tresnak hautatzea, eta datu-baseetarako sarbideen zehaztapenak interpretatzea.
  10. Talde-laneko testuinguruetan lidergo, motibazio, gainbegiratze eta komunikazioko teknikak garatzea taldekideen arteko komunikazioa errazteko.
  11. Garatutako software-osagaiak egiaztatzea, zehaztapenak aztertuta, proba-plana osatzeko.
  12. Moduluak instalatzea, eta horien egitura eta funtzionaltasuna aztertzea, aplikazio-zerbitzariak kudeatzeko.
  13. Tresna espezifikoak erabiltzea, ezarritako estandarrak betez, prozesuen dokumentazioari eustea.
  14. Segurtasun-beharrak identifikatzea eta aurrez ezarritako plana egiaztatzea, lotutako teknikak eta prozedurak aplikatzeko.
  15. Interfaze baten elkarreragina, irisgarritasuna eta erabilgarritasuna ebaluatzea, aplikazioaren interfazean multimedia-osagaiak integratzeko.
  16. Irisgarritasun unibertsalari eta guztiontzako diseinuari erantzuteko beharrezkoak diren lanbide-ekintzak identifikatzea eta proposatzea.

Ikaskuntzaren ebaluazioa (IE)

  1. Web interfaze baten sorrera planifikatzen du, diseinuko zehaztapenak baloratuta eta aplikatuta.
  2. Web interfaze homogeneoak sortzen ditu, estiloak definituz eta aplikatuz.
  3. Weberako multimedia-fitxategiak prestatzen ditu, horien ezaugarriak aztertuta eta berariazko tresnak erabilita.
  4. Web dokumentuetan multimedia-edukia integratzen du, ekarpena baloratzen du eta elementu elkarreragileak behar bezala hautatzen ditu.
  5. Web interfaze irisgarriak garatzen ditu, ezarritako jarraibideak aztertuta eta egiaztapen-teknikak aplikatuta.
  6. Web interfaze lagunkoiak garatzen ditu, ezarritako erabilgarritasun-jarraibideak aztertuta eta aplikatuta.
    7. --------------
  7. Garatu beharreko web aplikazioen interfazerako estilo-gida eta diseinua interpretatu ditu, diseinu-ekipoaren jarraibideak kontuan izanda.
  8. Web aplikazioen interfazea osorik edo partzialki osatzen duten markatzeko lengoaietan sententzia-blokeak editatu eta probatu ditu, estiloak kanpo-orrietatik administratuz.
  9. Informazioa bezeroaren aldean aurkezpen-formatu egokietara bihurtzeko eta egokitzeko transformazio-orriak erabili ditu.
  10. Web aplikazio baten interfazerako multimedia-materialak prestatzen eta integratzen parte hartu du, diseinu-ekipoaren jarraibideei jarraituz.
  11. Ekitaldiak maneiatzean eta elementu interaktiboak eta animazioak integratzean oinarritutako web aplikazio interaktiboak garatzen lagundu du.
  12. Web aplikazioen erabilerraztasuna eta erabilgarritasuna egiaztatu ditu, eta eskatutako mailak betetzeko beharrezko aldaketak eta neurriak egiten lagundu du.
  13. Web-aplikazioen zerbitzari-zatia diseinatzeko eta garatzeko arduradunekin lankidetza estuan aritu da, irizpideak bateratuz eta aplikazioaren bi aldeetako garapena koordinatuz.
  14. Aplikazioaren erabiltzaileari laguntzeko protokoloetan erabilitako dokumentazioa eta gainerako osagaiak definitzen eta lantzen parte hartu du.

Metodologia didaktikoa

Metodologia didaktikoa ikasgelan irakaste-ikaste prozesua bideratzeko hartzen diren erabakien multzoa da, betiere lanbide-moduluen ikaskuntzaren emaitzak lortzeko helburuarekin. Jardunbide horretan estrategia didaktikoak alde batetik, eta irakaste-ikaste jarduerak zehaztu behar ditugu.

Estrategia didaktikoek askotarikoak izan beharko dute, eta lantzen ari den ikaskuntzaren emaitzaren eta edukien sakontasunaren arabera bereizi beharko dira. Estrategia hauek erabiliko ditugu:

  • Ikasgaiaren azalpena.
  • Miaketa bibliografikoa.
  • Lan praktikoak.
Konstruktibismoa eta funtzionaltasuna ikaskuntzaren oinarri

Konstruktibismoa, ikasleei beren ezagutza-eskemak eraikitzen edo berregituratzen laguntzeko, lehendik dauzkaten eskemetan oinarrituta, ikaskuntza berriak ikasleen egitura psikologikoari begira eraikiz erakargarri suerta dakieken.

Funtzionaltasuna ikasitakoa beste testuinguru batzuetara esportatzen laguntzeko. Hori funtsezkoa baita bizi dugun lan-munduaren giro dinamikoan koka daitezen.

Printzipio metodologikoak

Ikuspegi kognitibo guztiek partekatzen dituzte printzipio metodologiko hauek:

  2. Abiapuntuko ezagutza-eskemak identifikatzea.
  3. Ikasleek zer eta zergatik egiten duten jakitea.
  4. Ikasleen jarduera sustatzea, motibazioa bultzatuz.

  6. Elkar onartzeko eta lankidetzan aritzeko giroa sortzea.

Irakaskuntza-ikaskuntza jarduerak

Irakaste-ikaste jarduerak hezkuntza-helburuak lortzera bideratuta egongo dira. Informazioa emateko prozesu antolatuak eta proposamen metodologikoak edo ikaskuntza-esperientziak gauzatzeko modu aktiboan landuko dira.

Diseinatu diren jarduerek honako xede hauek lortu behar dituzte:

  • Ikasleen artean jarduera bizia sustatzea.
  • Kontzientzian eta jokabide moralean heztea.
  • Modu kooperatiboan egituratzea, indibidualismoa eta lehiakortasuna alboratuta.

Ikasmaterialaren zeregina prozesu hori dinamizatzea izango da, sormena, lankidetza eta elkartasuna sustatuz.

Baina, helburu orokor horiek alde batera utzita, diseinatuko diren jarduerek Hezkuntza Sailaren Aginduetan zehazten diren ikaskuntzaren emaitzak garatzera bideratuta egon behar dute.

Emaitza horiek lortzeko jarduera koherenteak planteatuko dira orokortasunak baztertuta eta onartuz jarduera berak emaitza desberdinak ekar ditzakeela abiapuntuko aldagaiak aldatzean.

Ikasleekin erabiltzeko jarduera sorta zabala:
  • Aurkezpen-motibazioa: Ikasleak burutuko den prozesuan sartzeko eta edukien erabilgarritasuna iradokitzeko.
  • Aurretiazko ezagutzen ebaluazioa: Ikasgaiaren hasieran egingo dira, eta, beharrezkoa denean, ikasgai horren atal baten hasieran.
  • Edukien garapena: Ikaskuntzaren emaitzak eskuratzea dute helburu. Azalpen teorikoen, ebazpen praktikoen adibideen eta praktika gidatuen bidez egingo dira. Denbora gehiena hartuko dute.
  • Sendotzea: Ideia berriak aurrekoekin alderatuko dira eta ikaskuntza berriak eguneroko egoeretan eta testuinguru berrietan aplikatuko dira.
  • Laburpena: Ikasleei ideiak testuinguruan kokatzen eta irakasleei prozesuari buruzko informazioa lortzen laguntzeko.
  • Ebaluazioa: Lortutako helburuak eta lorpen-maila detektatzeko Hezkuntza Sailak emandako ebaluazio-irizpideetara egokituko dira. Unitate didaktiko guztietarako ebaluazio-jarduerak planifikatuko dira. Halaber, irakaste-ikaste prozesua bera ebaluatuko da.
  • Jarduera osagarriak: Saio horietako bakoitzean, ikasgelako jardueren bidez (ariketak eta/edo lanak) alderdi praktikoa lantzen edo ezagutzak finkatzen/handitzen saiatuko gara.

Ebaluazioa

Ebaluazioa hezkuntza-arloko esku-hartzerako tresna izango da, orientatuko gaituena eta egiten ari garen prozesuaren kalitatea baloratzeko aukera emango diguna. Irakaste-ikaste prozesua hobetzea du helburu.

Ebaluazioa jarraitua, orientatzailea eta integratzailea izango da.

Ebaluazio irizpideak

Ikaskuntza ebaluatzeko, goiana aipatutako ikaskuntzaren emaitzak (IE) eta modulu bakoitzarentzat finkatutako ebaluazio-irizpideak hartuko dira kontuan. Unitate didaktiko bakoitzean irizpideak zehaztuko dira, eta horien araberako ebaluazio-jarduerak programatuko.

Irakaskuntzaren ebaluazioa egiteko helburu didaktikoak, edukiak, baliabideak eta tenporizazioa aztertuko dira. Zeregin hori bloke bakoitzaren amaieran egingo da.

Sekuentziazioa

Ebaluazioa hiru fasetan egongo da sekuentziatuta:
  • Hasierakoa: ikasleari hurrengo faseetan aurkituko duenaren berri emateko. Ebaluazio hori medio, ikasleari ikasgaiaren laburpen moduko bat ematen zaio, galdera bidezkoa, zalantzak sortaraziz eta haien erantzuna bilatzeko motibazioa indartuz.
  • Jarraitua edo prestakuntzakoa: ikasleen aurrerapena eta darabilten lan-teknikak aztertu nahi dira, kontzeptu teorikoei buruzko galdetegiak eta proba teoriko-praktikoak erabiliz.
  • Amaierakoa: Prozesuaren fase bakoitza amaitzeko, ikaskuntzaren emaitzak baloratuko dira, betiere ebaluazio-irizpideen arabera.

Unitate didaktiko bakoitzak bere ebaluazio-jarduerak izango baditu ere, hainbat unitaterekin batera egin daitezke.

Era berean, ikasturtean zehar, multzo bateko edo batzuetako edukietan oinarrituta diseinatutako ariketa eta praktika globalak proposatuko dira behar besteko aurrerapenaz.

Zehar-lerroak

Lanbide-modulu hau garatzen den bitartean, ikasleengan honako hauekin lotutako jarrerak sustatzen saiatuko da:
  1. Gizonen eta emakumeen arteko erabateko berdintasuna lortzeko aurreikusitako neurriak aplikatuko dira irakasgaiaren garapenean lagundu duten emakumezko figurak nabarmenduz, berdintasunezko hizkuntza, inklusiboa eta ez-sexista erabiliz ahozko eta idatzizko azalpenetan eta sexuari lotutako estereotipoak saihestuz.
  2. Ingurumena zaintzeko hezkuntza bultzatuko da, papera eta tonerra ahalik eta gutxien erabiltzeko ahalegina sustatuz eta, erabiltzen bada, haiek birziklatzea eta berrerabiltzea balioetsiz.
  3. Hezkuntza morala eta gizatiarra, errespetuzko jarrera eta hizkuntza erabiliz ahozko zein irudizko azalpenetan.
  4. Osasunerako hezkuntza, ergonomiaren eta jarrera-ohituren bidez, eta saioetan egindako iradokizunen bidez.

Lanabesak eta baliabide didaktikoak

Irakaskuntza-ikaskuntza prozesuan, jarduera bakoitzerako egokiak diren baliabideetan oinarrituko gara kontuan hartuta bitartekoak direla eta ez helburuak.

Erabiliko diren baliabideen hautaketa eduki motaren, helburuen eta ikasleen ikasteko erritmoaren araberakoa izango da. Jakina, ebaluazio-irizpideekin bat etorri beharko dute.

Ikasgaiak, helburuak, ebaluazioak

1.- Web-bezeroen gaineko programazio-teknologiak eta -arkitekturak hautatzen ditu, bakoitzaren gaitasunak eta ezaugarriak identifikatuta eta aztertuta.
Ebaluazio-irizpideak:
  1. Zerbitzarian eta web bezeroan kodea exekutatzeko ereduak ezaugarritu eta bereizi ditu.
  2. Web nabigatzaileen gaitasunak eta kodea exekutatzeko mekanismoak identifikatu ditu.
  3. Web bezeroen programazioarekin lotzen diren lengoaia nagusiak identifikatu eta ezaugarritu ditu.
  4. Gidoien programazioaren berezitasunak ezagutu ditu, bai eta programazio tradizionalaren gainean dituen abantailak eta desabantailak ere.
  5. Markatzeko lengoaiak web bezeroen programazio-lengoaiekin integratzeko mekanismoak egiaztatu ditu.
  6. Web bezeroen gaineko programazio-tresnak ezagutu eta ebaluatu ditu.

2.- Epai sinpleak idazten ditu, hizkuntzaren sintaxia aplikatuta eta web nabigatzaileen gainean exekutatzen dela egiaztatuta

Ebaluazio-irizpideak:
  1. Web bezeroak programatzeko lengoaia hautatu du, aukeren arabera.
  2. Hizkuntzan erabilgarri dauden aldagai eta eragile motak erabili ditu.
  3. Aldagaien erabilera-esparruak identifikatu ditu.
  4. Hizkuntzaren berezitasunak ezagutu eta egiaztatu ditu, datu moten arteko bihurketei dagokienez.
  5. Epaien blokeak sortzeko erabaki-mekanismoak erabili ditu.
  6. Kodeari iruzkinak gehitu dizkio.
  7. Begiztak erabili ditu eta funtzionamendua egiaztatu du.
  8. Kodearen programazioa, proba eta arazketa errazteko tresnak eta inguruneak erabili ditu.

3.- Kodea idazten du, eta, horretarako, lengoaiaren objektu aurredefinituek ekarritako funtzionaltasunak identifikatzen eta aplikatzen ditu.

Ebaluazio-irizpideak:
  1. Lengoaiaren objektu aurredefinituak identifikatu ditu.
  2. Nabigatzailearen leihoei dagozkien objektuak eta horien web-dokumentuak aztertu ditu.
  3. Lengoaiaren objektu aurredefinituak nabigatzailearen itxura eta jasotzen duen dokumentua aldatzeko erabiltzen dituzten sententziak idatzi ditu.
  4. Nabigatzailean kodea gauzatzearen ondorioz testuak eta etiketak sortu ditu.
  5. Lengoaiaren objektu aurredefinituak erabiltzailearekin elkarreragiteko erabiltzen dituzten sententziak idatzi ditu.
  6. Hainbat leihoz eta markoz osatutako dokumentuetan hizkuntzaren berezko ezaugarriak erabili ditu.
  7. Cookieak erabili ditu informazioa biltegiratzeko eta edukia berreskuratzeko. h) Kodea araztu eta dokumentatu du.

4.- Web bezeroentzako kodea programatzen du, erabiltzaileak definitutako egiturak aztertuz eta erabiliz.

Ebaluazio-irizpideak
  1. Lengoaiaren funtzio aurredefinituak sailkatu eta erabili ditu.
  2. Erabiltzaileak definitutako funtzioak sortu eta erabili ditu.
  3. Arrayak sortzeari eta erabiltzeari lotutako lengoaiaren ezaugarriak ezagutu ditu.
  4. Array-ak sortu eta erabili ditu.
  5. Hizkuntzaren objektuetara orientatzeko ezaugarriak ezagutu ditu.
  6. Objektuen egitura definitzeko kodea sortu du.
  7. Metodoak eta propietateak sortu ditu.
  8. Erabiltzaileak definitutako objektuak erabiltzen dituen kodea sortu du.
  9. Kodea araztu eta dokumentatu du.

5.- Web aplikazio elkarreragileak garatzen ditu, gertaerak maneiatzeko mekanismoak integratuta.

Ebaluazio-irizpideak
  1. Sortutako gertaerak atzitzeari dagokionez, markatzeko lengoaiaren aukerak ezagutu ditu.
  2. Ekitaldien kudeaketari buruzko programazio-lengoaiaren ezaugarriak identifikatu ditu.
  3. Maneiatu daitezkeen gertaera motak bereizi ditu.
  4. Gertaerak harrapatu eta erabiliko dituen kodea sortu du.
  5. Web inprimakiak kudeatzeko lengoaiaren gaitasunak ezagutu ditu.
  6. Web formularioak balidatu ditu ekitaldiak erabilita.
  7. Baliozkotze-prozedurak errazteko adierazpen erregularrak erabili ditu.
  8. Kodea probatu eta dokumentatu du.

6.- Web aplikazioak garatzen ditu, dokumentuko objektuen ereduaren ezaugarriak aztertu eta aplikatuta.

Ebaluazio-irizpideak
  1. Web orri bateko dokumentuaren objektuen eredua ezagutu du.
  2. Modeloaren objektuak, propietateak eta metodoak identifikatu ditu.
  3. Dokumentuaren egiturara sartzen den kode bat sortu eta egiaztatu du.
