Ikasgaiaren atalaren gida didaktikoa (HTML)

Ikasgaia oso-osorik HTML markatzeko lengoaiari eskainia dago. HTMLk dokumentuaren barruko espazioa definitzen du. Bertan kokatzen dira, eduki estatikoa eta dinamikoa, eta bera da aplikazio guztietarako oinarrizko plataforma. Webgune edo web aplikazio bat sortzeko, lehenik eta behin, hura osatzen duten orrialdeetako bakoitzaren egitura zehazten duen HTML kodea idatzi behar dugu.

Gure dokumentuen edukia HTML kode gisa behar bezala interpretatzen dela ziurtatzeko, fitxategiaren hasieran erantsi behar dugu adierazpena. Adierazpen hori HTML etiketen antzekoa da, eta dokumentu bakoitzaren hasieran eskatzen da, nabigatzaileari web-orria nola sortu behar duen erabakitzen laguntzeko. HTML5ekin programatutako dokumentuen kasuan, deklarazioak html atributua barne hartu behar du: <!DOCTYPE html>.

Saioan hiru atal bereizi dira:

  • Egitura: HTML elementuek zuhaitz moduko egitura bat osatzen dute, elementua sustraitzat hartuta. Egitura horrek antolamendu-maila ugari ditu, eta elementu batzuk dokumentuaren atal orokorrak definitzeaz arduratzen dira, eta beste batzuk atal txikiak edo edukia irudikatzeaz. Egituraren bizkarrezurra definitzeko eta nabigatzaileak orria pantailan erakusteko hainbat 'elementu' eskaintzen ditu.

  • Edukia: Gure webgunearen oinarrizko egitura amaituta, edukia landu behar dugu. Orain arte aztertutako HTML elementuek diseinuaren atal bakoitza identifikatzen eta helburu bat esleitzen laguntzen digute, baina web-orri batean benetan axola duena atal horien barruan dagoena da. Informazio hori hainbat elementu bisualez osatuta dagoenez (izenburuak, testuak, irudiak eta bideoak, besteak beste), HTMLk hura irudikatzeko hainbat elementu definitzen ditu.

  • Formularioak: Hauek goikoak bezala edukia dira webguneetan, baina hainbestekoa denez euren garrantzia atal bereizian erakutsi ohi dira. Formularioak dokumentu batean sar ditzakegun tresnak dira, erabiltzaileek informazioa txertatu, erabakiak hartu, datuak komunikatu eta aplikazio baten portaera aldatzeko aukera izan dezaten. Formularioen helburu nagusia da erabiltzaileak informazioa hautatu edo txertatu ahal izatea eta zerbitzarira bidaltzea, prozesatzeko.

Gauzak horrela atal hau amaitzean ikaslea gai izango da:

  • Edukiak web aplikazio baten logikan integratzeko eta zehaztapenetara egokitutako datuak atzituko dituzten osagaiak garatzeko.

  • Web aplikazioetan interfazeak garatzeko estilo-eskuliburuaren arabera, markatzeko lengoaiak eta web-estandarrak erabilita.

  • Web aplikazioetan integratuko diren multimedia-osagaiak erabiltzeko ezarritako zehaztapenak medio.

  • Web aplikazio baten interfazean multimedia-osagaiak integratzeko.
