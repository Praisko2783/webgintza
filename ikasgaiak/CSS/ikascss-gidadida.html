Atalaren gida didaktikoa (CSS) (EGITEKE, EGITEKE)

CSS1 hasierako bertsiotik, CSS2 bilakaeratik eta egungo CSS3 estandarretik, ur-jauzian dauden estilo-orrien estandarra (CSS) sakontasunez aurkezten da.

  • o Teknologia horien konbinazioak web multimedia bezero erakargarriak eta indartsuak garatzea ahalbidetzen du.
  • o Diseinu malguaren kontzeptuak sartzen dira.
  • o Web erabilerraztasunaren oinarrizko kontzeptuak, oinarrizko osagaiak eta egungo estandarrak sartzen dira.

Gure dokumentua HTML5ean modu egokian antolatuta eta idatzita daukagularik, apainketarekin hasteko moduan izango gara. Horretarako CSS erabiliko dugu. Atal honetan estilo lengoaia horren xehetasunak aurkeztuko ditugu. Orri eder eta erakargarri bat osatzeko gutxuieneko xehetasunak. Atal honekin batera, beste bi ere prestatuko ditugu:
Web-orriak pantaila tamaina ezberdinetara moldatzeko, diseinu moldagarria egiten lagunduko diguten 'media queries'ak ikasteko bat eta
web-orriko elementuak antolatzeko bi elementu garrantzitsu, flex eta grid nola erabili ikasteko eta kaxek web-orrian daukaten berezitasunak aztertzeko.

Saioan hiru atal bereizi dira:

  • Estiloak: Estiloak atxitzen ikasiko dugu, urjauzi orriaren kontzeptua zertan den eta adibide batzuk medio zer-nola gauzatzen den pantailan.
    Bideoa: Idatziz emandakoa modu praktikoan erakutsiko da ikusentzunezko batean.
    Azterketa: Irakatsitakoa indartzeko azterketatxoa egingo da, ikusitakoaren gainean.

  • Erreferentziak: Izenak, Atributuak eta Seudoklaseak nola erabili behar diren eta zertarako ikasiko dugu.
    Bideoa: Idatziz emandakoa modu praktikoan erakutsiko da ikusentzunezko batean.
    Azterketa: Irakatsitakoa indartzeko azterketatxoa egingo da, ikusitakoaren gainean.

  • Propietateak: Testua / Kolorea / Tamaina / Atzealdea izango dira lehen azpiatalean landuko diren ikasgaiak. Ertzak / Itzalak / Gradienteak / Iragazkiak bigarren azpiatalean landuko ditugu. Eraldaketak / Trantsizioak / Animazioak hirugarrenean. Azpiatal horietako bakoitzak bere ikus-entzunezkoa izango du idatziz emandakoa modu praktikoan ikusteko.
    Atala amaitzeko azterketatxoa egingo dugu ikasitakoa eguneartzeko.

Gauzak horrela atal hau amaitzean ikaslea gai izango da:

  • Edukiak web aplikazio baten logikan integratzeko eta zehaztapenetara egokitutako datuak atzituko dituzten osagaiak garatzeko.

  • Web aplikazioetan interfazeak garatzeko estilo-eskuliburuaren arabera, markatzeko lengoaiak eta web-estandarrak erabilita.

  • Web aplikazioetan integratuko diren multimedia-osagaiak erabiltzeko ezarritako zehaztapenak medio.

  • Web aplikazio baten interfazean multimedia-osagaiak integratzeko.
