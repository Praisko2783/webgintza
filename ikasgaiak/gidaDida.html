Gaitasun orokorrak

Web Aplikazioen Garapena, Goi-mailako Lanbide Heziketako titulazio bat da. Titulazio horren helburu nagusia web aplikazioak garatzea, ezartzea eta mantentzea da, erabilitako metodoa edozein delarik baina teknologia espezifikoak erabiliz, datuak modu seguruan atzitzea bermatuz eta ezarritako estandarretan exijitutako erabilerraztasuneko, usagarritasuneko eta kalitateko irizpideak betez.

Gaitasun profesionalak, pertsonalak eta sozialak

Titulazio honen jabea gai izango da:
  • Informatika-sistemak konfiguratzeko eta ustiatzeko, eta, horretarako, sistemaren konfigurazio logikoa doitzeko, ezarritako irizpideen eta erabilera-beharren arabera.
  • Sistema, zerbitzu eta aplikazioetako segurtasunarekin lotzen diren teknikak eta prozedurak aplikatzeko, segurtasun-plana beteta.
  • Aplikazio-zerbitzariak kudeatzeko, horien konfigurazioa kasuen arabera egokituta, eta, hala, web aplikazioen garapena eta hedapena ahalbidetzeko.
  • Datu-baseak kudeatzeko, eta, horretarako, horien diseinu logikoa interpretatzeko eta datuen osotasuna, konsistentzia, segurtasuna eta eskuragarritasuna egiaztatzeko.
  • Datu-baseetarako sarbidea duten web aplikazioak garatzeko, zehaztapenetara egokitutako lengoaiak, atzipen-objektuak eta mapaketa-tresnak erabilita.
  • Edukiak web aplikazio baten logikan integratzeko eta zehaztapenetara egokitutako datuak atzituko dituzten osagaiak garatzeko.
  • Web aplikazioetan interfazeak garatzeko estilo-eskuliburuaren arabera, markatzeko lengoaiak eta web-estandarrak erabilita.
  • Web aplikazioetan integratuko diren multimedia-osagaiak garatzeko, eta, horretarako, tresna espezifikoak erabiltzeko eta ezarritako zehaztapenak betetzeko.
  • Web aplikazio baten interfazean multimedia-osagaiak integratzeko, eta aplikazioaren interaktibitatea, erabilerraztasuna eta usagarritasuna analizatzeko.
  • Web zerbitzariaren ingurunean software osagaiak garatzeko eta integratzeko, tresna eta lengoaia espezifikoak erabilita, eta, hala, aplikazioaren zehaztapenak betetzeko.
  • Zerbitzuak garatzeko eta haien funtzioak beste web aplikazio batzuetan integratzeko, funtzionaltasuna ziurtatuta.
  • Web aplikazioetan zerbitzu eta eduki banatuak integratzeko, eta haien funtzionaltasuna ziurtatzeko.

Irakaskuntza hauek laneko arriskuen prebentzioko oinarrizko jarduerak gauzatzeko beharrezko ezagupenak hartzen dituzte barnean.

Heziketa-zikloaren helburu orokorrak:

  1. Markatzeko lengoaiak eta web-estandarrak erabiltzea, web-aplikazioetan interfazeak garatzeko.
  2. Lengoaiak, objektuak eta tresnak erabiltzea, eta garatzeko zehaztapenak interpretatzea.
  3. Lengoaiak, objektuak eta tresnak hautatzea, eta datu-baseetarako sarbideen zehaztapenak interpretatzea.
  4. Talde-laneko testuinguruetan lidergo, motibazio, gainbegiratze eta komunikazioko teknikak garatzea taldekideen arteko komunikazioa errazteko.
  5. Garatutako software-osagaiak egiaztatzea, zehaztapenak aztertuta, proba-plana osatzeko.
  6. Moduluak instalatzea, eta horien egitura eta funtzionaltasuna aztertzea, aplikazio-zerbitzariak kudeatzeko.
  7. Tresna espezifikoak erabiltzea, ezarritako estandarrak betez, prozesuen dokumentazioari eustea.
  8. Segurtasun-beharrak identifikatzea eta aurrez ezarritako plana egiaztatzea, lotutako teknikak eta prozedurak aplikatzeko.
  9. Interfaze baten elkarreragina, irisgarritasuna eta erabilgarritasuna ebaluatzea, aplikazioaren interfazean multimedia-osagaiak integratzeko.
  10. Irisgarritasun unibertsalari eta guztiontzako diseinuari erantzuteko beharrezkoak diren lanbide-ekintzak identifikatzea eta proposatzea.

Ebaluazio-irizpideak

  1. Garatu beharreko web aplikazioen interfazerako estilo-gida eta diseinua interpretatu ditu, diseinu-ekipoaren jarraibideak kontuan izanda.
  2. Web aplikazioen interfazea osorik edo partzialki osatzen duten markatzeko lengoaietan sententzia-blokeak editatu eta probatu ditu, estiloak kanpo-orrietatik administratuz.
  3. Informazioa bezeroaren aldean aurkezpen-formatu egokietara bihurtzeko eta egokitzeko transformazio-orriak erabili ditu.
  4. Web aplikazio baten interfazerako multimedia-materialak prestatzen eta integratzen parte hartu du, diseinu-ekipoaren jarraibideei jarraituz.
  5. Ekitaldiak maneiatzean eta elementu interaktiboak eta animazioak integratzean oinarritutako web aplikazio interaktiboak garatzen lagundu du.
  6. Web aplikazioen erabilerraztasuna eta erabilgarritasuna egiaztatu ditu, eta eskatutako mailak betetzeko beharrezko aldaketak eta neurriak egiten lagundu du.
  7. Web-aplikazioen zerbitzari-zatia diseinatzeko eta garatzeko arduradunekin lankidetza estuan aritu da, irizpideak bateratuz eta aplikazioaren bi aldeetako garapena koordinatuz.
  8. Aplikazioaren erabiltzaileari laguntzeko protokoloetan erabilitako dokumentazioa eta gainerako osagaiak definitzen eta lantzen parte hartu du.

Urdinez eta azpimarraz zehaztu ditut proiektuaren lehen joaldi honetan zuzen-zuzenean ukitu asmo ditudan moduluak. Egia da beste moduluetako batzuk (urdinez, azpimarra gabe) zeharka bada ere, aipatu aipatuko direla, eta askotan landu ere bai, hala nola Datu Baseak, Programazioa, Web aplikazioen hedapena..., baina ez dira izango proiektuaren zati honen helburu.

Zeharkako horiek 'Web-garapena zerbitzari-ingurunean' izenaz bataiatu asmo dudan proiektuaren bigarren joaldian landuko dira zuzen-zuzenean, eta honako honetan zuzen-zuzenean jorratuko direnak izango dira orduan, zeharka landuko direnak.

Web garapena bezeroaren ingurunean

Azpiatalak

Ikaskuntzaren emaitzak eta ebaluazio-irizpideak.

1.- Web-bezeroen gaineko programazio-teknologiak eta -arkitekturak hautatzen ditu, bakoitzaren gaitasunak eta ezaugarriak identifikatuta eta aztertuta.
Ebaluazio-irizpideak:
  1. Zerbitzarian eta web bezeroan kodea exekutatzeko ereduak ezaugarritu eta bereizi ditu.
  2. Web nabigatzaileen gaitasunak eta kodea exekutatzeko mekanismoak identifikatu ditu.
  3. Web bezeroen programazioarekin lotzen diren lengoaia nagusiak identifikatu eta ezaugarritu ditu.
  4. Gidoien programazioaren berezitasunak ezagutu ditu, bai eta programazio tradizionalaren gainean dituen abantailak eta desabantailak ere.
  5. Markatzeko lengoaiak web bezeroen programazio-lengoaiekin integratzeko mekanismoak egiaztatu ditu.
  6. Web bezeroen gaineko programazio-tresnak ezagutu eta ebaluatu ditu.

2.- Epai sinpleak idazten ditu, hizkuntzaren sintaxia aplikatuta eta web nabigatzaileen gainean exekutatzen dela egiaztatuta

Ebaluazio-irizpideak:
  1. Web bezeroak programatzeko lengoaia hautatu du, aukeren arabera.
  2. Hizkuntzan erabilgarri dauden aldagai eta eragile motak erabili ditu.
  3. Aldagaien erabilera-esparruak identifikatu ditu.
  4. Hizkuntzaren berezitasunak ezagutu eta egiaztatu ditu, datu moten arteko bihurketei dagokienez.
  5. Epaien blokeak sortzeko erabaki-mekanismoak erabili ditu.
  6. Kodeari iruzkinak gehitu dizkio.
  7. Begiztak erabili ditu eta funtzionamendua egiaztatu du.
  8. Kodearen programazioa, proba eta arazketa errazteko tresnak eta inguruneak erabili ditu.

3.- Kodea idazten du, eta, horretarako, lengoaiaren objektu aurredefinituek ekarritako funtzionaltasunak identifikatzen eta aplikatzen ditu.

Ebaluazio-irizpideak:
  1. Lengoaiaren objektu aurredefinituak identifikatu ditu.
  2. Nabigatzailearen leihoei dagozkien objektuak eta horien web-dokumentuak aztertu ditu.
  3. Lengoaiaren objektu aurredefinituak nabigatzailearen itxura eta jasotzen duen dokumentua aldatzeko erabiltzen dituzten sententziak idatzi ditu.
  4. Nabigatzailean kodea gauzatzearen ondorioz testuak eta etiketak sortu ditu.
  5. Lengoaiaren objektu aurredefinituak erabiltzailearekin elkarreragiteko erabiltzen dituzten sententziak idatzi ditu.
  6. Hainbat leihoz eta markoz osatutako dokumentuetan hizkuntzaren berezko ezaugarriak erabili ditu.
  7. Cookieak erabili ditu informazioa biltegiratzeko eta edukia berreskuratzeko. h) Kodea araztu eta dokumentatu du.

4.- Web bezeroentzako kodea programatzen du, erabiltzaileak definitutako egiturak aztertuz eta erabiliz.

Ebaluazio-irizpideak
  1. Lengoaiaren funtzio aurredefinituak sailkatu eta erabili ditu.
  2. Erabiltzaileak definitutako funtzioak sortu eta erabili ditu.
  3. Arrayak sortzeari eta erabiltzeari lotutako lengoaiaren ezaugarriak ezagutu ditu.
  4. Array-ak sortu eta erabili ditu.
  5. Hizkuntzaren objektuetara orientatzeko ezaugarriak ezagutu ditu.
  6. Objektuen egitura definitzeko kodea sortu du.
  7. Metodoak eta propietateak sortu ditu.
  8. Erabiltzaileak definitutako objektuak erabiltzen dituen kodea sortu du.
  9. Kodea araztu eta dokumentatu du.

5.- Web aplikazio elkarreragileak garatzen ditu, gertaerak maneiatzeko mekanismoak integratuta.

Ebaluazio-irizpideak
  1. Sortutako gertaerak atzitzeari dagokionez, markatzeko lengoaiaren aukerak ezagutu ditu.
  2. Ekitaldien kudeaketari buruzko programazio-lengoaiaren ezaugarriak identifikatu ditu.
  3. Maneiatu daitezkeen gertaera motak bereizi ditu.
  4. Gertaerak harrapatu eta erabiliko dituen kodea sortu du.
  5. Web inprimakiak kudeatzeko lengoaiaren gaitasunak ezagutu ditu.
  6. Web formularioak balidatu ditu ekitaldiak erabilita.
  7. Baliozkotze-prozedurak errazteko adierazpen erregularrak erabili ditu.
  8. Kodea probatu eta dokumentatu du.

6.- Web aplikazioak garatzen ditu, dokumentuko objektuen ereduaren ezaugarriak aztertu eta aplikatuta.

Ebaluazio-irizpideak
  1. Web orri bateko dokumentuaren objektuen eredua ezagutu du.
  2. Modeloaren objektuak, propietateak eta metodoak identifikatu ditu.
  3. Dokumentuaren egiturara sartzen den kode bat sortu eta egiaztatu du.
  4. Egituraren elementu berriak sortu ditu eta lehendik zeuden elementuak aldatu ditu.
  5. Ekintzak ereduko gertaerekin lotu ditu.
  6. Ereduak hainbat nabigatzailetan dituen desberdintasunak identifikatu ditu.
  7. Web aplikazioak programatu ditu, ereduaren hainbat inplementazio dituzten nabigatzaileetan funtzionatzeko moduan.
  8. Web aplikazioetan hiru alderdiak (edukia, itxura eta portaera) independizatu ditu.

7.- Web aplikazio dinamikoak garatzen ditu, eta, horretarako, bezeroaren eta zerbitzariaren arteko komunikazio asinkronoko mekanismoak ezagutzen eta aplikatzen ditu.

Ebaluazio-irizpideak
  1. Bezeroaren eta web zerbitzariaren arteko komunikazio asinkronoko mekanismoak erabiltzearen abantailak eta eragozpenak ebaluatu ditu.
  2. Komunikazio asinkronoa ezartzeko erabilgarri dauden mekanismoak aztertu ditu.
  3. Erlazionatutako objektuak erabili ditu.
  4. Propietateak eta metodoak identifikatu ditu.
  5. Komunikazio asinkronoa erabili du web-dokumentuaren eguneratze dinamikoan.
  6. Informazioa bidaltzeko eta jasotzeko hainbat formatu erabili ditu.
  7. Web aplikazio asinkronoak programatu ditu, hainbat nabigatzailetan funtzionatzeko moduan.
  8. Web-orrien programazioan eguneratze dinamikoko teknologiak sartzea erraztuko duten liburutegiak sailkatu eta aztertu ditu.
  9. Liburutegi horiek erabiltzen dituzten programak sortu eta araztu ditu.
