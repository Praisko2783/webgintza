Atalaren gida didaktikoa (JavaScript) (EGITEKE, EGITEKE)

Unitate didaktiko honetan, ikasleek oinarrizko kontzeptuak ikasiko dituzte. JavaScript programazioa eta HTML5 eta CSS eta Canva aplikatuko dituzte joko sinplea sortzeko. Proiektu praktiko honen bidez, problemak ebaztea, pentsamendu logikoa eta sormena..

Ikasleek landuko dituzten gaitasunak

  • Gaitasun digitala: Gaitasunak eta trebetasunak bereganatuko dituzte Informazioaren eta Komunikazioaren Teknologiak (IKT) sortzeko eduki digitala.
  • Matematikarako gaitasuna eta zientzia eta teknologiako oinarrizko gaitasunak: Kontzeptu matematikoak eta logikoak aplikatuko dituzte jokoa gauzatzeko.
  • Ikasten ikasteko gaitasuna: Ikaskuntza autonomorako eta arazoak konpontzeko estrategiak garatuko dituzte jokoaren programazioan erronkei aurre egiten dioten heinean.
  • Ekimenaren zentzuzko gaitasuna eta espiritu ekintzailea: Beren sormena eta berrikuntzarako gaitasuna landuko dituzte beren jokoa diseinatzean eta garatzean.

Helburuak

  • JavaScripteko programazioaren oinarrizko kontzeptuak ulertzea, aldagaiak, kontrol-egiturak, funtzioak eta gertaerak maneiatzea barne.
  • Erabili HTML5 Canvas elementua web orri batean grafikoak eta animazioak sortzeko.
  • Joko sinple bat diseinatzea eta garatzea, JavaScript eta Canvas erabiliz.
  • Pentsamendu logikoa eta problemak programazioaren bidez ebaztea sustatzea.
  • Sormena eta adierazpen pertsonala sustatzea jokoaren diseinuan
