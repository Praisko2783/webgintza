Kokapen geografikoa

API Geolocation diseinatu zen detekzio mekanismo estandar bat eskaintzeko garatzaileek kokapen geografikoa zehazteko erabiltzen dituzten nabigatzaileak erabiltzailea. Lehen, datu-base zabal bat eraikitzeko aukera baino ez genuen, IP helbideak eta zerbitzarian kontsumo handiko aplikazioak programatzea, soilik ematen zizkigutenak erabiltzailearen kokapenari buruzko gutxi gorabeherako ideia bat (batzuetan herrialdea bezain zehaztugabea). API honek sistema berriak baliatzen ditu, hala nola sareen triangelazioa eta GPSa, aplikazioa exekutatzen ari den gailuaren kokapena itzultzeko.

Lortutako informazioa erabiltzailearen kokapenera egokitzen diren aplikazioak sortzeko erabiltzea edo informazioa ematea automatikoki lokalizatua. APIan hiru metodo erabiltzen dira helburu horrekin.

  • getCurrentPosition (kokapena, errorea, konfigurazioa) — Metodo hau banakako eskaerak. Hiru propietate onartzen ditu: lortutako kokapena prozesatzeko funtzio bat,akatsak prozesatzeko funtzio bat eta objektu bat informazioa nola lortuko den konfiguratzeko(lehenengo atributua baino ez da behar).
  • watchPosition (kokapena, errorea, konfigurazioa) — Metodo hau aurreko metodoaren antzera jokatzen du. Alñde bakarra da metodo honek kokapena etengabe kontrolatzen duela, gertatutako edozein aldaketaren berri emanez.
  • setInterval () — Metodo honek prozesua denbora tarte batean automatikoki errepikatzen du lehenetsitako balioen arabera edo atributuek zehaztutako konfigurazioaren arabera.
  • ClearWatch (identifikatzailea) — Metodo honek watchPosition metodoa ()k hasitako prozesua geldiarazi egiten du. clearInterval () metodoaren antzekoa da. Azken horrek setInterval()ek hasitako prozesua geldiarazten du.

Hurrengoa izango da kapitulu honetarako gure dokumentua. Dokumentu erraza da, elementu baten barruan botoi bat baino ez dugu erabiliko < section> lokalizazio-sistemak lortzen duen informazioa.


<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="utf-8">
    <title>Geolocation</title>
    <script src="geolocation.js"></script>
  </head>
  <body>
    <section id="ubicacion">
      <button type="button" id="obtener">Obtener mi ubicacion</button>
    </section>
  </body>
</html>

Kokapen geografikoa

Lehen aipatu dugun bezala, atributu bakarra esleitu behar diogu getCurrentPosition () metodoari kokapena lortzeko. Atributu hori jasotzen duen funtzioa da Position izeneko objektu bat, lokalizazio-sistemak lortzen duen informazio guztiarekin.

Position objektuak bi propietate ditu balioak biltegiratzeko.
