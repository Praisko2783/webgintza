Ikasgaiaren atalaren gida didaktikoa (Webguneak)

Web-garatzaileek hainbat programazio-lengoaia erabiltzen dituzte bezeroen beharrak erabat asetzen dituzten webguneak egiteko. Esku artean dueun proiektuaren garrantziaren arabera, hainbat programatzaile arduratu daitezke Web Garapenaren alderdi ezberdinez, eta taldean lan egin behar dute proiektua aurrera ateratzeko.

HTMLren ehunka lerro txikitzeaz gain, Web Garatzaileek bilerak izan behar dituzte bezeroekin eta diseinatzaileekin, haien helburuak ezagutzeko eta ulertzeko, webgunearen funtzioak probatzeko aurrerapenak egin ahala, behin martxan jarri ondoren webgunearen errendimendua zein den ikusteko eta, are gehiago, etengabeko mantentze-lanak egiteko.

Garrantzitsua da ahalegin handia inbertitzea eta programazio-lengoaia pare bat gutxienez menderatzea, baita Web Garapeneko bi prozesu ezagunenekin ohitzea ere: Waterfall (ur-jauzian garatzea) eta Agile.

Ikasgaia oso-osorik webgune gehienak osatzen duten elementuen berri ematea du ehlburu nagusi. HTMLk dokumentuaren barruko espazioa definitzen du. Bertan kokatzen dira, eduki estatikoa eta dinamikoa, eta bera da aplikazio guztietarako oinarrizko plataforma. Webgune edo web aplikazio bat sortzeko, lehenik eta behin, hura osatzen duten orrialdeetako bakoitzaren egitura zehazten duen HTML kodea idatzi behar dugu.

Saioan hiru atal bereizi dira:

  • Webguneak: Webgune bat funtsezko tresna bihurtu da mundu digitalean presentzia indartu nahi duen edozein pertsona edo enpresarentzat. Negozio txiki bat, korporazio handi bat edo profesional independente bat izan, web orri bat izatea da Interneten ikusgarritasuna irabazteko modurik eraginkorrenetako bat.
    Atal honetan, sakonago aztertuko dugu zer diren webguneak, eta nola funtzionatzen duten ulertuko dugu, haien osagaiak, motak eta kategoriak aztertuz.

  • Lengoaiak: Lehen ikusi dugun bezala, web zerbitzari batek web orri bat, estilo orri bat edo script bat eman ditzake, besteak beste. Horiek sortzeko, web lengoaiak erabiltzen dira: HTML, CSS eta JavaScript, hiru lengoaia garrantzitsuenak.

  • Lanabesak: Web garatzaile gisa, web aplikazio fidagarriak sortzeko ardura duzu. Horrek, programazioa ez ezik, zeregin konplexuak ere eskatzen ditu, denbora asko kontsumitzen dutenak, hala nola erroreak garbitzea eta zerbitzariak administratzea.
    Zorionez, programazio-tresnek prozesu osoa sinplifikatzen lagun dezakete, kalitatea arriskuan jarri gabe. Tresna horiek ikusiko ditugu atal honetan.

Gauzak horrela atal hau amaitzean ikaslea gai izango da:

  • Interneten gure datuak nola mugitzen diren ikuskatzeko.

  • Webgunea argitara emateko beharrezkoak diren domeinua eta ostalaria zer diren eta nola kontratatzen diren ezagutzeko.

  • Bezeroa eta Zerbitzaria kontzeptuak ongi bereizteko eta haiek garatzeko erabiltzen diren lengoaiak antzemateko.

  • Webgunea bat garatzeko behar diren gutxieneko lanabesak (Ostalari lokala, Kode editorea, Navigatzaileak, Bertsioen kontrola...) erabiltzeko.
